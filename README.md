# Easy Flight free game on Godot and C#

Gameplay:
- Complete 5 levels.
- Defeat 5 bosses and get all the upgrades.
- Get access to the endless mode.
- Score as many points as possible and have fun!

## Installation

- Install Godot 4+.
- Install Net SDK.
- Install IDE like JB Rider.
